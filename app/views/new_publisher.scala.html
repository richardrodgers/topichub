Define your publisher profile

Choose a category that best describes the type of publisher, or 'other' if none fits.

Fill in the fields at left making sure that you select a 'pubId' that:

  • is not already in use on the hub.
  • is a short handle containing no spaces
  • is mnemonically helpful

Put the full name in the name field, and add a terse description in the next one.

Add a URL to your main site in the link field, and a URL for a logo if you have one.
